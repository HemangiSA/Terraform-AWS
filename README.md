## Prerequisites

- AWS Account
- AWS CLI
- Terraform
- Git
- Visual Studio Code

## Configure AWS

```bash
aws configure
```

## Initialize Terraform

```bash
terraform init
```

## Validate

```bash
terraform validate
```

## Plan

```bash
terraform plan
```

## Apply

```bash
terraform apply
```

## Destroy

```bash
terraform destroy
```
